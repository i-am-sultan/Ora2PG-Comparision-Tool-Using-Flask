# Flask Data Validation Application

## Overview
This Flask application allows users to validate data from Oracle and PostgreSQL databases by comparing row counts of specified tables.

## Prerequisites
- Python 3.x installed on your system: https://www.python.org/downloads/
- Access to Oracle and PostgreSQL databases
- Required Python packages listed in `requirements.txt`
- To get all the requirement module just type "pip install -r requirements.txt"

## Installation
1. Clone or download this repository to your local machine.
2. Navigate to the project directory in your terminal.

## Setup
1. Install dependencies by running: "pip install -r requirements.txt"
2. Configure database credentials:
- Open `app.py` and modify the database connection settings (Oracle and PostgreSQL) as per your environment.

## Usage
1. Run the Flask application:
2. Open a web browser and go to `http://localhost:5000` to access the application.
3. Enter Oracle and PostgreSQL database credentials along with the table name to validate.
4. Click on the "Validate" button to compare row counts from both databases.
